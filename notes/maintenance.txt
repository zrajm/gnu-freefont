1)  Using Change Log mode in Emacs

    C-x 4 a adds a new entry to the change log file

2)  Commiting a CVS file

    export CVS_RSH=ssh
    cvs -z3 -d peterlin@savannah.nongnu.org:/cvsroot/freefont co freefont
    cd freefont/sfd
    cvs -z3 -d peterlin@savannah.nongnu.org:/cvsroot/freefont commit FreeSans.sfd

3)  Makefile

    "make" creates two compressed tar files: one with sources (SFD
    files) and one with TrueType fonts. The naming convention is:

    freefont-sfd-yyyymmdd.tar.gz
    freefont-ttf-yyyymmdd.tar.gz

    yyyymmdd is the year, month and date of release.

4)  Creating detached signature file

    gpg -b freefont-ttf-20060126.tar.gz

    This creates a file named freefont-ttf.tar.gz.sig. Option -ba
    creates an ASCII signature (with suffix .asc); Savannah
    documentation however requires binary detached signature (see
    https://savannah.gnu.org/faq/?admin=&group_id=5802&question=How_do_I_add_files_in_the_download_area.txt)

5)  Adding files to the download area

    * ftp savannah.gnu.org
    * cd incoming/savannah/freefont
    * mput freefont-ttf-20060126.tar.gz freefont-ttf-20060126.tar.gz.sig
